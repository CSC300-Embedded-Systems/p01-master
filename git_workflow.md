# How to use git

## How to get started

1. Install git
2. Open Git Bash
3. Clone the repo:
  a. ```git clone \<repo_URL\>```
  b. cd \<repo_dir\>

## Start coding

1. git branch \<branchname\>
2. git checkout \<branchname\>
3. \<write code\>
4. git add -A or git add \<filename\>
5. git commit -m "\<witty message\>"
6. git push 

Your files are now in Github.
